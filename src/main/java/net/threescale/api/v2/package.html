This is the version2 API for the 3scale backend. It is used to encapsulate the communication with the 3scale backend server for validating and reporting transactions between the user and the provider of a service.

The version 2 api include caching of authorize operations and batching of responses sent to the server.

For the definitive specification of the interface refer to The 3scale Management API specification

Example code

In the package net.threescale.api.v2 there is an example of how to call the api.

Jar file

The runtime Api for both version 1 and 2 is in the distribution tarball in dist/threescale-api.jar.
